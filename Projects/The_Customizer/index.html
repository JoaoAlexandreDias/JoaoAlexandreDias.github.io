The Customizer

This project was my first experience with the programming language p5.js.

The objective of the project was to create a way to display time in an alternative way. As such, I decided to create a clock that mimicked the design of a few digital watches.

As an additional challenge, I decided to implement customization options that are available to larger screen sizes.
